Copy this prompt and paste it into ChatGPT, Claude, or any AI assistant to generate a complete quiz JSON: 

Act as an expert educator and quiz creator. Create a quiz about [INSERT YOUR TOPIC HERE].

- For "multiple_choice", use "correct_answers" with an array of correct indices.
- Provide both "question" (English) and "question_ar" (Arabic) fields.
- Provide both "options" (English) and "options_ar" (Arabic) arrays.
- explanation and explanation_ar are optional but recommended.
- Keep "image" as an empty string unless you have a valid image URL.
- Generate at least 5 questions for a complete quiz experience.
