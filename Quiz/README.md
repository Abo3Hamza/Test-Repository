## 📌 Overview

**Pro Quiz Master** is not just a static webpage; it is a highly scalable, data-driven **Quiz Engine**. Built entirely with Pure Logic and Vanilla JavaScript, it allows educators and developers to deploy complex, interactive quizzes instantly. The entire UI is generated dynamically based on a single JSON file, meaning you can create entirely new tests without touching a single line of HTML or CSS.

---

## ✨ Key Features

### 🧠 Engine Intelligence
* **Multi-Format Support:** Handles 3 different question types seamlessly: True/False, Single Choice, and Multiple Choice (with advanced validation logic).
* **Instant Freezing:** Once a question is answered, options are instantly locked to ensure test integrity and prevent tampering.
* **Smart Auto-Skip:** Navigating to the next question without answering automatically flags the question as "Skipped," allowing users to return and solve it later.

### 🎨 Immersive User Experience (UX)
* **Dynamic Theme Switcher:** A floating menu offering **15 distinct themes**, including professional Dark Modes, allowing users to customize their environment.
* **Real-Time Translation:** Full bilingual support. Toggle between English and Arabic instantly for questions, options, and explanations.
* **Fully Responsive:** Flawless rendering across all devices (Desktop, Tablet, Mobile).

### ⏱️ Advanced Time Management
* **Three Timer Modes:** 1. *Untimed:* For relaxed studying and review.
  2. *Count-Up (Stopwatch):* To track how long the quiz takes.
  3. *Count-Down (Challenge):* Strict time limits set in minutes.
* **Timeout Handling:** Automatically resolves the quiz when the time is up, marking all remaining questions as skipped and compiling the final report.

### 📊 Results & Analytics
* **Comprehensive Summary:** Displays the final percentage, total time taken, and a detailed breakdown of answers (Correct, Wrong, Skipped).
* **Mistakes Review:** A dedicated section detailing incorrect answers, revealing the correct choice alongside a rich explanation.
* **Skipped Review:** Highlights unanswered questions with their respective solutions and explanations.

## 🚀 How to Use (Data-Driven Architecture)

This repository is designed as a **Template**. You do *not* need to edit the HTML to create your own quiz.

1. Open the `questions.json` file.
2. Add your questions, options, and explanations following the structure provided in `JSON_STRUCTURE.md`.
3. Push the updated JSON file to your repository.
4. *Done!* The JavaScript engine will automatically fetch the data, parse it, and render your custom quiz dynamically.

---

## 🛠️ Technical Details

Built with a philosophy of **Logic over Libraries**:
* **Frontend:** HTML5, CSS3 for modern layout and animations.
* **Core Logic:** Vanilla JavaScript (No React, Vue, or jQuery required).
* **Data Management:** JSON for seamless data structuring and fetching.
